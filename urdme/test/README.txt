README for urdme/test/

-- spin - medium level tests
 Navigate to /urdme/test in Matlab and execute
   spin or spin(N)
 to run all spin tests or the Nth one

-- scrub - extended level tests

-- spin_aem - medium level tests (AEM solver)
-- spin_ssa - medium level tests (SSA solver)
-- spin_uds - medium level tests (UDS solver)

-- spin_subdiffusion - medium level tests (SUBDIFFUSION workflow)
-- scrub_subdiffusion - extended level tests (SUBDIFFUSION workflow)

-- scrub_neuron - extended level tests (NEURON workflow)

-- scrub_DLCM - extended level tests (DLCM workflow)

-- examples - tests taken from the examples folder

-- URDMEtest - run all of the above tests
